# ClipForge AI

Plataforma web completa inspirada no Opus Clip para gerar cortes automáticos de lives e vídeos com IA.

## Stack
- Frontend: Next.js 15 + React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Banco de dados: PostgreSQL + Prisma ORM
- Autenticação: JWT + Google OAuth
- IA: OpenAI, Whisper e FFmpeg

## Estrutura
- frontend/: interface moderna e responsiva
- backend/: API modular com rotas para upload, processamento e histórico
- services/: lógica de IA e processamento

## Como executar
1. Instale o Node.js 20+
2. Execute `npm install`
3. Inicie o projeto com `npm run dev`

> Em ambientes sem Node.js instalado, a execução local não será possível até instalar o runtime.
